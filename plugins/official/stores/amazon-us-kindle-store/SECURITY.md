# Security Policy

Report security issues for this plugin privately when possible. Prefer the [BookAtrium security policy](https://github.com/lgdeysel1980/BookAtrium/blob/main/SECURITY.md) and repository security reporting tools.

This plugin:

- Uses BookAtrium's host-provided HTTP service.
- Declares the Amazon hosts it needs.
- Does not request credentials, API keys, cookies, proxy settings, or affiliate tags.
- Does not purchase, download, or decrypt Kindle content.
- Does not embed Amazon pages or automate a browser.

Please do not include private ebooks, account credentials, cookies, access tokens, or purchase details in public issues.
