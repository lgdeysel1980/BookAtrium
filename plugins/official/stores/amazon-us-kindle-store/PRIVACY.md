# Privacy

Amazon US Kindle Store does not collect personal information, telemetry, analytics, or Amazon credentials.

When the user searches, the search terms entered in BookAtrium are sent directly to Amazon.com through BookAtrium's plugin HTTP service. Amazon receives normal request information under Amazon's own policies.

When the user activates a result, BookAtrium opens the Amazon product page in the user's default browser. Any account activity, browsing, purchasing, or privacy choices happen on Amazon's website and are outside this plugin.

The plugin does not:

- Request or store Amazon usernames or passwords.
- Access Amazon accounts, Kindle libraries, orders, carts, or wish lists.
- Download Kindle content.
- Handle or remove DRM.
- Send search terms or results to PractiCore or BookAtrium.
- Use telemetry, analytics, affiliate tags, cookies, or custom tracking.
