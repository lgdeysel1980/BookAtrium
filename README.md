# BookAtrium

BookAtrium is a Windows ebook library, reader, organiser, and management application designed to make managing a personal ebook collection straightforward and enjoyable.

This repository is the official public home for BookAtrium downloads, documentation, support, feature requests, and software issue reporting.

> The BookAtrium application source code is maintained separately and is not stored in this public repository.

## Download BookAtrium

The latest stable version of BookAtrium will be available from the official GitHub Releases page.

[Download the latest BookAtrium release](../../releases/latest)

Only download BookAtrium from this repository or another official BookAtrium download location.

## Reporting a Problem

When reporting a problem, please include:

- Your BookAtrium version
- Your Windows version
- The affected area of the application
- A clear description of what happened
- The steps needed to reproduce the problem
- The result you expected
- Any relevant error message
- Screenshots where useful

Do not include passwords, licence keys, private ebooks, personal documents, database backups, or other confidential information.

## Feature Requests

Feature requests are welcome.

Please explain:

- What problem the feature would solve
- How the feature would improve BookAtrium
- An example of how you would use it
- Any alternatives or workarounds you currently use

Early ideas and general feedback can also be discussed in [GitHub Discussions](../../discussions).

## Releases and Updates

Stable versions of BookAtrium will be published through GitHub Releases.

- The signed BookAtrium installer

BookAtrium may also use the official release information in this repository to check for application updates.

## Supported Platform

BookAtrium is currently being developed for:

- Windows 10
- Windows 11
- 64-bit systems

Additional platforms may be considered in the future.

## Security

Please do not use public Issues to report a security vulnerability that could place users or their data at risk.

Security reporting instructions will be added before the first public release.

## Privacy

BookAtrium support reports should never contain:

- Passwords
- Licence keys
- Personal identification information
- Private ebook files
- Customer or business information
- Unedited diagnostic files containing sensitive paths or data

Always review screenshots, logs, and diagnostic information before uploading them publicly.
